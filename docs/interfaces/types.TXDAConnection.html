  • requestCurrentDesign3d(): void

    Request the latest current design be dispatched with its 3D structure (if any). This is more expensive to fetch than TXDAConnection.requestCurrentDesign, therefore it is not automatically dispatched on design edits and changes, and must be requested with this method.

    Current design data with the 3D structure can be handled with TXDAMessageHandlers.onUpdateCurrentDesign3d.
